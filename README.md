天罡圖(v1.0)是在圖片裡面加入一個密碼加密的文件，從而使用圖片傳遞加密文件的方法。

使用方法是： 首先選擇jpg圖片（大的比較好），然後選擇要加密的文件，之後輸入密碼，最後點擊"加料"。

提取方法是： 首先選擇天罡圖，然後輸入密碼，最後點擊"提取"

生成的文件名是自動選取的，生成天罡圖是在原圖名後加-tgt.jpg. 提取的解密文件是圖名加.secret
要改變缺省輸出目錄可點擊右邊小按鈕。

運行狀態給出 提示信息。

注意：提取時密碼錯誤不會提示，會得到不可讀文件。
